== Description ==

Let you include word definitios or explanations, in a tool-tip style. The explanation appears when 
the user pass the mouse over a tagged word.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the directory inside the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.

Using in your post

5. Go to some post that you have already writed.
6. For create a tagged tool-tip word, write the next sintax:
   [textToBeUnderlined|TextThatWillAppearInToolTip]
   
7. By example 
   [wordpress|Popular web blog system]
   [NFL|National Football Ligue]
   [cephalalgia|is a condition of pain in the head; sometimes neck or upper back pain]
